check installed or not

Here’s the quickest way to check if Python is installed on your system:

🖥️ On Windows
Open Command Prompt

Press Win + R, type cmd, hit Enter

Type this command:

python --version
✅ What you should see
✔ If installed correctly:

Python 3.11.x
❌ If NOT installed or not added to PATH:

You may see:

'python' is not recognized...
or it opens the Microsoft Store

🔄 If it doesn’t work
Try:

python3 --version
If still not working, it likely means:
👉 You didn’t check “Add Python to PATH” during installation

🧪 Bonus check (optional)
You can also try:

pip --version
If pip shows a version → Python is fully set up 🎉
